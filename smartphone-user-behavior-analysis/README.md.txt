# Smartphone User Behavior Analysis

## Project Overview

This project analyzes smartphone user behavior data to understand factors that influence user engagement and app usage patterns.

The analysis focuses on relationships between operating systems, screen time, demographics, and app usage behavior.

## Dataset

The dataset contains **700 smartphone users** with the following features:

* Device Model
* Operating System
* App Usage Time (minutes per day)
* Screen On Time (hours per day)
* Battery Drain
* Data Usage
* Age
* Gender
* User Behavior Class

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Project Questions

1. Which operating system has higher average app usage?
2. Does screen time correlate with app usage?
3. How does age affect smartphone engagement?
4. What user behavior segments exist?

## Key Insights

* Android users show slightly higher average app usage.
* Screen time strongly correlates with app engagement.
* Younger users tend to spend more time using mobile apps.
* High engagement users belong to behavior class 4.
